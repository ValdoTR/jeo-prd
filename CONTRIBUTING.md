# Contributing

PRs welcome! 

## Quick Start

1. Fork the repo
2. Create a branch (`feature/thing` or `fix/thing`)
3. Make your changes
4. Test with `/jeo-prd` in Claude Code
5. Open a PR with a short description

## Guidelines

- Keep it simple — this is a small skill
- Test your changes before submitting
- One PR per feature/fix

## Questions?

Open an issue.
